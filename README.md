# AIDOS – AI Desktop Operating System

## Overview

AIDOS is an AI-powered desktop operating assistant capable of understanding natural language commands and automating desktop and browser workflows.

---

## Features

- AI Chat
- Desktop Automation
- Browser Automation
- File Organization
- Memory
- Dashboard
