# 🌍 Overview

AIDOS (AI Desktop Operating System) is an intelligent desktop assistant designed to simplify computer interactions through natural language.

Instead of manually performing repetitive tasks, users can describe what they want in plain English, and AIDOS intelligently plans and executes the required workflow using specialized AI agents.

The project combines artificial intelligence, desktop automation, browser automation, memory systems, and a scalable backend into a modern desktop application.
